## Thèmes de skins (lancement)

### 🤖 Thème 1 — Industrial (default, livré avec le jeu)
- Style brut, rouille, kaki militaire
- Inspiration : Wall-E, Mad Max
- Inclus pour tous les joueurs

### 🌸 Thème 2 — Sakura Bots
- Robots décorés de motifs floraux japonais
- Couleurs pastel rose / blanc / or
- Inspiration : street fashion Harajuku
- **Skin event spring 2026**

### 👻 Thème 3 — Spectral
- Robots translucides, effets fantomatiques
- Couleurs bleu glacial / violet
- Inspiration : Hollow Knight, Spiritfarer
- **Skin event Halloween 2026**

### 🎮 Thème 4 — Retro 8-bit
- Robots façon NES / Game Boy
- Effets pixelisés et sons rétro
- Inspiration : nostalgie console
- **Skin event meta-référence**

### 🔥 Thème 5 — Inferno
- Robots métalliques rouges, effets de feu
- Inspiration : Doom, Hellboy
- **Skin event "Forge des âmes"**

### 🦊 Thème 6 — Animal Companions
- Robots stylisés en animaux (chat, renard, ours)
- Inspiration : Adventure Time, Studio Ghibli
- **Hook narratif fort** pour Option B "Robot avec une âme"

## Acquisition (comment on obtient des skins)

### 🆓 F2P paths
1. **Progression naturelle** : 1 skin commun par milestone (niveau 5, 10, 20…)
2. **Daily challenges** : 1 skin garanti par semaine (rare ou + selon perf)
3. **Battle Pass gratuit** : 3-5 skins par saison
4. **Events** : skins thématiques limités via tokens d'event

### 💰 Paid paths
1. **Boutique direct** : skins à l'unité (200-5 000 gems)
2. **Bundles** : -30% à -50% par rapport à l'achat individuel
3. **Battle Pass premium** : 9,99€/saison, ~15-20 skins
4. **Drop boxes éthiques** : voir contenu avant d'ouvrir, **pas de duplicata**

### ⚠️ Règles d'or
- **Aucun skin n'est jamais 100% loot box random** (pas de gambling caché)
- **Tous les skins sont visibles dans le catalogue** (même les hors-stock)
- **Pas de pay-to-win** : skins n'affectent JAMAIS les stats
