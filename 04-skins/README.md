# 04 — Skins & Customisation

## Pourquoi ce dossier ?

Les **skins** sont la **#1 source de monétisation éthique** sur mobile en 2026.

- ✅ Pas de pay-to-win
- ✅ Expression personnelle des joueurs
- ✅ Source de marketing organique (screenshots, TikTok)
- ✅ Source de revenu **récurrent** via skins limités / saisonniers

## Principes directeurs

1. **Jamais de pay-to-win** : skins = pur cosmétique
2. **Lisibilité gameplay préservée** : un robot doit rester identifiable même skinné
3. **Diversité de thèmes** : pas que cyberpunk → fantasy, animaux, anciens, espace…
4. **FOMO modéré** : skins limités OK mais pas tous (besoin de skins permanents pour F2P)
5. **Cohérence d'univers** : chaque skin doit respecter le lore (Option B "Robot avec une âme")
