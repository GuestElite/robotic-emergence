## 🥇 Top références gameplay (à analyser en profondeur)

### Mushroom Wars 2 (Zillion Whales, 2017)
**À voler** :
- 📌 Système de **lignes de progression** (souris drag)
- 📌 UI propre sur mobile
- 📌 Skins de factions

### Plants vs Zombies (PopCap, 2009)
**À voler** :
- 📌 **Tutoriel exemplaire** (le meilleur du mobile)
- 📌 **Onboarding progressif** des unités
- 📌 Mode **Endurance** bien calibré
- 📌 Mini-jeux entre niveaux

### Mindustry (Anuken, 2017)
**À voler** :
- 📌 **Système de chemins / conveyors** (notre signature gameplay)
- 📌 Tower defense + factory hybride
- 📌 Modes survival
- 📌 Génial pour ce qui est de connecter des bâtiments

### Clash Royale (Supercell, 2016)
**À voler** :
- 📌 Auto-battle après placement
- 📌 **Sessions ultra-courtes** (3 min max)
- 📌 **Système de coffres** (loot)
- 📌 Économie F2P maitrisée
- 📌 **Esports** scene

### Swords & Soldiers (Ronimo Games, 2009)
**À voler** :
- 📌 Lane unique side-scroll
- 📌 Combat humoristique
- 📌 Factions très différentiées

## 🥈 Références méta-progression / rétention

### Vampire Survivors (poncle, 2022)
**À voler** :
- 📌 **Méta-progression exemplaire** (gold ↔ stat boosts perma)
- 📌 Loop "die, upgrade, run again"
- 📌 Variety via reliques/unlocks
- 📌 0 monétisation au début → on n'est pas obligés d'IAP-le

### Slay the Spire (Mega Crit, 2017)
**À voler** :
- 📌 **Méta-progression par classes**
- 📌 Daily challenges avec modificateurs
- 📌 Ascensions = difficulté progressive

### Hades (Supergiant, 2020)
**À voler** :
- 📌 **Mort = progression** (chaque échec apporte qqch)
- 📌 Narration par fragments
- 📌 Heat system (modificateurs de difficulté)

### Brotato (Blobfish, 2022)
**À voler** :
- 📌 **Build variety** énorme
- 📌 Daily run gratuit
- 📌 Persos à débloquer

### Risk of Rain 2 (Hopoo Games, 2020)
**À voler** :
- 📌 **Difficulté qui scale dans le temps**
- 📌 Items synergistiques
- 📌 Loops perfectibles

## 🥉 Références monétisation / LiveOps

### Genshin Impact (miHoYo, 2020)
**À voler** :
- 📌 **Events bi-hebdo de qualité**
- 📌 Battle Pass mensuel
- 📌 F2P-friendly avec IAP whaling
- 📌 **Story dripping** mensuel

### Marvel Snap (Second Dinner, 2022)
**À voler** :
- 📌 **3 min max** par partie
- 📌 Battle Pass clair
- 📌 Onboarding magistral
- 📌 Esthétique nostalgique

### Coin Master (Moon Active, 2015)
**À voler** :
- 📌 **Engagement loop hardcore** (3+ retours/jour)
- 📌 Push notif perfectionnés
- 📌 Social asynchrone (raid amis)
⚠️ Mais : éthique borderline, ne pas copier le côté dark patterns

### Royal Match (Dream Games, 2021)
**À voler** :
- 📌 **First-time UX brillant**
- 📌 Loading screens utiles
- 📌 Pas de friction à l'IAP

## 🎨 Références art direction

### Loop Hero (Four Quarters, 2021)
**À voler** :
- 📌 Pixel art moderne **lisible sur petit écran**
- 📌 Animation économique
- 📌 Palette restreinte mais marquée

### Mini Metro (Dinosaur Polo Club, 2014)
**À voler** :
- 📌 **Minimalisme total**
- 📌 Couleurs codées par fonction
- 📌 Sound design exceptionnel

### Battle Chasers Nightwar (Airship Syndicate)
**À voler** :
- 📌 Comic-style 2D
- 📌 Animation fluide

### Reigns (Nerial, 2016)
**À voler** :
- 📌 Style cards minimaliste
- 📌 Tons SF/médiéval

### Necropolis (Harebrained Schemes)
**À voler** :
- 📌 **Robots stylisés rétro**
- 📌 Ambiance néon-noir
