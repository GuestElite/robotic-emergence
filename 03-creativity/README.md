# 03 — Créativité & Brainstorms

## Objectif

Espace **libre et désordonné** pour balancer des idées sans contrainte. Pas de validation, pas de format strict. Le but est de **générer beaucoup** pour ensuite **trier**.

## Règles

- ✅ Aucune idée trop dingue
- ✅ Pas besoin de tout justifier
- ✅ On peut contredire les autres dossiers
- ✅ On capture, on tri plus tard

## Contenu

| Fichier | Description |
|---------|-------------|
| [brainstorm-mechanics.md](brainstorm-mechanics.md) | Idées de mécaniques nouvelles |
