## Principes directeurs (éthique commerciale)

1. **F2P doit pouvoir tout débloquer gratuitement** (juste plus lentement)
2. **Aucun pay-to-win** sur les stats / talents
3. **Transparence** : tous les drops visibles, pas de RNG caché
4. **Pas de FOMO toxique** : pas de "Plus que 30 secondes !" sur les offres
5. **Refund-friendly** : tu te trompes d'achat, refund OK
6. **Pas de loot box gambling**
