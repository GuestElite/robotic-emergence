# 01 — Vision & Pitch

## Objectif du dossier

Définir l'identité du jeu **avant** de spec le moindre détail :
- Quelle est notre **histoire** ? (pivot vs Animatrix)
- Quel est le **nom** ?
- Quel est notre **angle de différenciation** ?

## Contenu

| Fichier | Statut |
|---------|--------|
| [differentiation.md](differentiation.md) | ⚠️ Lecture obligatoire — les 4 axes de différenciation |
| [pitch-options.md](pitch-options.md) | 🟡 **À DÉCIDER** — 3 pivots narratifs proposés |
| [working-name.md](working-name.md) | 🟡 **À DÉCIDER** — shortlist de noms |
| `pitch-final.md` | ⏳ À créer après tes choix |

## Workflow

1. Lis `differentiation.md` → cadre nos contraintes
2. Lis `pitch-options.md` → choisis 1 pitch (ou propose un 4e)
3. Lis `working-name.md` → shortlist 3 noms
4. Une fois validé : je rédige `pitch-final.md` (1 page)
