# 06 — Événements & LiveOps

## Pourquoi ce dossier ?

**LiveOps** = ce qui fait vivre le jeu **après le lancement**. Sans events réguliers, ta D30 retention chute drastiquement.

> *Règle empirique mobile : ce que tu fais en LiveOps détermine 80% de ton revenu à 12 mois.*

## Principes directeurs

1. **Fréquence > intensité** : mieux vaut 2 petits events que 1 énorme/an
2. **Pas tous payants** : 70% F2P-friendly, 30% boost premium possible
3. **Réutilisation** : mêmes mécaniques skinées différemment = économie de prod
4. **Calendrier publié** : transparence vs surprises (préférence joueurs 2026)
5. **Themes saisonniers** : Halloween, Noël, Saint-Valentin, été…
